### 제목

`Type: 제목`

#### Type의 종류
```text
Feat: 새 기능 추가
Fix: 버그 수정
Docs: 문서 수정
Style: 코드 포맷 변경, 세미콜론 누락, 코드 수정이 없는 경우
Refactor: 프로덕션 코드 리팩터링
Test: 테스트 추가, 테스트 리팩터링(프로덕션 코드 수정 없음)
Chore: 빌드 태스크 업데이트, 패키지 매니저 설정의 경우
```
